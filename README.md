# Online-Excel-Analystics-Platform

A powerful full-stack web platform that allows users to upload any Excel file (.xls or .xlsx), analyze the data, and generate interactive 2D and 3D charts in real time.
Built using the MERN stack, this project combines data visualization, analytics, and AI-powered insights in one seamless interface.
